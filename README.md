
# Android Basics User Interface

An introduction to UI ( User-Interface ).

### Language Required

XML

### XML Syntax

```
<ClassName
    attribut1
    attribut2
    attribut3 />
```

#### Example

- TextView

```javascript
<TextView
        android:text="Hello World!!"
        android:background="@android:color/darker_gray"
        android:layout_width="150dp"
        android:layout_height="75dp"
        />
```
